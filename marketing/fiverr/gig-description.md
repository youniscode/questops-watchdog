# Gig Description

## I will set up Discord monitoring for your self-hosted game server

Self-hosted game servers crash, fill up disks, and stop producing logs. Without monitoring, you find out when your players message you. With QuestOps Watchdog, you get Discord alerts the moment something goes wrong -- process stops, disk fills up, logs go stale, backups stop running.

I will install, configure, and test a lightweight PowerShell-based watchdog on your Windows machine. Everything runs locally. No cloud accounts, no monthly subscriptions, no databases, no Python.

## What you receive

- A working watchdog installation that checks your server every few minutes
- Discord alerts when something fails, with severity colors (red = critical, orange = warning, green = recovered)
- Config validation so your setup is correct before any check runs
- Scheduled task integration for fully automated monitoring
- Maintenance mode: suppress alerts during game updates without disabling anything
- Recovery alerts: one notification when a problem resolves itself, then silence
- Optional grouped summary at the end of each run

## Supported game servers

Project Zomboid, Valheim, Minecraft, ICARUS, 7 Days to Die, Enshrouded, Conan Exiles, Palworld, and any Windows dedicated server with a process name, log folder, or backup folder.

## What I need from you

- Remote desktop or screen share access to your Windows machine
- Your Discord server with permission to create a webhook
- The folder paths where your game server files live
- The name of your server process (visible in Task Manager)

## What I do not need

- Admin rights (basic operation works without them)
- Your Discord webhook URL written anywhere in a config file
- Any software installation (PowerShell is built into Windows)

## What this service does not include

- Cloud or SaaS setup
- Dashboard or web interface
- Auto-healing or server restarts
- 24/7 support or uptime guarantees
- Modifications to your game server files

## Delivery format

You receive a configured ZIP containing everything needed to run, plus documentation and a walkthrough of your setup.
